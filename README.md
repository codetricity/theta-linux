# RICOH THETA USB API on Jetson Nano

Install guide.

## How to Run This Document Locally

Uses MkDocs and Material MkDocs.  

## Installation

```bash
    $ pipenv install
    $ pipenv install mkdocs
    $ pipenv install mkdocs-material
    $ pipenv install mkdocs-material-extensions
    $ pipenv shell
    $ mkdocs serve
```
